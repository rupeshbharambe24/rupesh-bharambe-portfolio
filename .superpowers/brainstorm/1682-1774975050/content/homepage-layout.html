Key Design Decisions

  • Monospace labels on stats (hackathons.won) — techy touch without going full terminal
  • Gradient logo mark (RB) — replaced with your real photo in production
  • Typing effect on tagline — rotates through: "intelligent systems", "AI solutions", "real-world impact"
  • Stats bar — animated counters that count up when scrolled into view
  • Featured projects — only top 3, color-coded by category, links to full Projects page
  • Two-column teaser — About + Skills side-by-side, both link to deeper pages

Framer Motion Animations on this page

  • Hero: photo scales in → name fades up → tagline starts typing → buttons stagger in
  • Stats: counter animation (0 → 5, 0 → 15+, etc.) triggered on scroll
  • Project cards: stagger reveal from bottom, tilt + glow on hover
  • About/Skills: slide in from left/right respectively
  • Page transitions: shared layout animation between pages
