Design Decisions

  • Stats summary bar at top — animated counters, each with unique accent color + subtle gradient background
  • Trophy wall — 5 win cards in 2-column grid, each with unique border color + radial glow. HackCelestial spans full width as the 5th card
  • Other honors — compact list rows for SIH 2025, Avishkar Silver, Pune Agri International, with expandable "show more" for remaining finalists
  • Certifications — clean 3-column grid with issuer icons, only the 7 high-signal certs

Animations

  • Stats: counter animation (0 → 5, 0 → 15+, etc.) on page load
  • Trophy cards: stagger reveal with scale-in + subtle glow pulse
  • Honor rows: slide-in from left, staggered
  • Cert cards: fade-up in grid pattern
