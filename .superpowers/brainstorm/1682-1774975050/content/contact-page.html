Design Decisions

  • Two-column layout — form on left (wider), info + socials on right
  • Monospace labels on form fields — consistent techy feel
  • "Open To" card — colored badges showing what you're available for, recruiters see this instantly
  • Social links — branded icon boxes with platform colors (LinkedIn blue, Discord purple, IG gradient)
  • GitHub contribution graph — themed in purple (matches theme accent), fetched via GitHub API, shows you're actively coding
  • Footer — minimal, shows "Built with Next.js + Framer Motion" as a subtle tech signal

Animations

  • Form: fields stagger reveal from top to bottom
  • Right column: cards slide in from right, staggered
  • Social links: hover → slight lift + glow in platform brand color
  • GitHub graph: squares fade in one-by-one in a wave pattern (the "wow" moment on this page)
  • Submit button: magnetic effect on hover (like your old portfolio)
