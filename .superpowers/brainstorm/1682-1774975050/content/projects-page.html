Card Design Patterns

  • Featured project (MOSDAC) — full-width, horizontal layout with preview area + details + metrics
  • Standard cards — category badge + hackathon-win badge (if applicable) + description + metrics + tech + links
  • Hackathon winner badge — orange trophy badge on cards that won competitions
  • Metrics — bold stat numbers with monospace labels (accuracy, latency, data scale)
  • Filter chips — animated with layoutId so active filter slides smoothly (Framer Motion)

Animations

  • Cards: stagger reveal on page load, masonry-style cascade
  • Filter switching: AnimatePresence — cards exit/enter with layout animation
  • Card hover: subtle lift (translateY -4px) + border glow in theme accent color
  • Metrics: counter animation when card enters viewport
